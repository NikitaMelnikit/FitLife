# FitLife

FitLife is a modern, minimalist fitness and sports application designed to elevate your training reality. It provides customized workouts, intelligent nutrition tracking, and precise progress analytics.

## Features

- **Smart Workouts**: Tailored exercise routines designed for specific goals and fitness levels, ranging from Hypertrophy to HIIT Cardio Blasts.
- **Precision Nutrition**: Personalized meal plans and an advanced macro calculator (TDEE) to track your daily intake (Calories, Protein, Fats, Carbs).
- **Data-Driven Progress**: Visualize your transformation with beautiful, intuitive charts and logs that monitor your body weight over time.
- **Sleek Minimalist Interface**: A precision-crafted dark theme with high-contrast neon accents for maximum visibility and aesthetic appeal.
- **Real-Time Synchronization**: Powered by Firebase to keep your workout logs and progress metrics synced seamlessly.

## Technologies Used

- **Frontend**: React 19, Vite, Tailwind CSS 4
- **Backend & Database**: Firebase (Authentication, Firestore, Analytics)
- **Data Visualization**: Recharts
- **Icons & Animations**: Lucide React, Motion

## Getting Started

1. Clone the repository.
2. Install the dependencies using `npm install`.
3. Start the development server using `npm run dev`.

## Project Structure

- `/src/components`: Reusable UI components (Navbar, Footer, Buttons).
- `/src/contexts`: React contexts for state management (Authentication).
- `/src/pages`: Main application views (Home, Workouts, Nutrition, Calculator, Progress).
- `/src/lib`: Utility functions and Firebase configuration.
